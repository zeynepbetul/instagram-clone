# Instagram Clone

This project is a simple Instagram Clone developed using Swift. Users can sign up, log in, upload photos with comments, view posts from all users, and like posts. The app uses Firebase Authentication, Firebase Firestore, and Firebase Storage to manage users, data, and media in real time.

## Features

- User registration and login (Firebase Authentication)
- Upload photos from the photo library
- Store uploaded images in Firebase Storage
- Save post data (image URL, comment, email, date, like count) in Firestore
- Display all posts in a real-time feed
- Like button functionality with Firestore updates
- Images displayed with SDWebImage (cached & optimized)
- Posts ordered by date (newest first)
- MVC architecture

## Technologies Used

- Swift (UIKit)
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- SDWebImage
- UITableView
- UIImagePickerController
- MVC Architecture
